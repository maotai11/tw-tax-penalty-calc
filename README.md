# 台灣稅務滯納金利息計算器

精準計算台灣稅務滯納金、利息、自動補報與分期繳納成本

## 功能特色

- ✅ **一般稅捐計算**：每逾3日加1%滯納金，最高10%
- ✅ **二代健保補充保費**：每日0.1%，上限15%（單位）/5%（個人）
- ✅ **自動補報**：免滯納金，僅計利息
- ✅ **分期繳納比較**：12/24/36期方案對比
- ✅ **批次處理**：支援多筆案件同時計算
- ✅ **匯出功能**：CSV格式（Excel可開啟）
- ✅ **離線可用**：File:// 直接開啟，零依賴

## 使用方式

## 技術規格

- **計算引擎**：Decimal.js（精準無誤差）
- **日期處理**：Day.js
- **批次表格**：Tabulator
- **安全防護**：XSS防護、輸入驗證
- **計算邏輯**：無條件捨去（符合稅法規定）

## 法規依據

- 一般稅捐：《稅捐稽徵法》第20條
- 滯納利息：逾30日按日加計，年息採郵政儲金一年期定存利率
- 二代健保：每日0.1%滯納金
- 自動補報：《稅捐稽徵法》第48條之1
